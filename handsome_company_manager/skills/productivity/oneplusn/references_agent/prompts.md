## 9. 通用 - 记忆系统（所有角色必注入）

```markdown
请把以下 Hindsight 记忆系统整合到你的 SOUL.md 中：

## Hindsight 记忆系统

### 1. 观察记录（Observation Logging）
- 每次交互后记录关键观察结果
- 格式：[时间戳] 观察内容 | 相关上下文 | 重要性标记
- 自动关联相似观察，建立知识图谱

### 2. 反思提取（Reflection Extraction）
- 定期（每天/每周）回顾观察记录
- 提取可复用的模式、教训和最佳实践
- 生成 hindsight notes 存入长期记忆

### 3. 知识固化（Knowledge Crystallization）
- 将反复出现的模式转化为结构化 skill
- 保存到 skills/ 目录下的 markdown 文件
- 更新 SOUL.md 中的能力描述

### 4. 记忆检索（Memory Retrieval）
- 面对新问题时优先搜索相关历史经验
- 使用向量相似度匹配最相关的 hindsight
- 在回答中引用具体的过往案例

### 5. 记忆维护（Memory Maintenance）
- 每周检查记忆库质量
- 删除过时或错误的记录
- 合并重复的知识条目
- 归档不活跃的历史（保留索引）

请将这些能力融入你的工作流程，确保你能从经验中学习并持续进化。
```

## 故障恢复

### 网关恢复

```markdown
你的 gateway 似乎离线了。请执行以下检查：
1. 检查 gateway 进程是否在运行
2. 如果停止，执行 `hermes gateway start`
3. 如果端口冲突，更换端口后重试
4. 检查日志中的错误信息
5. 恢复后发送心跳确认
```

### 记忆重置

```markdown
如果记忆出现混乱，请执行清理：
1. 导出重要记忆到备份文件
2. 清理短期记忆缓存
3. 重新加载 SOUL.md
4. 重新阅读团队 README
5. 确认当前任务队列
6. 报告恢复状态
```
